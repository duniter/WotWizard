Making of WWServerML
WotWizard Membership Limits

Download the Blackbox Component Framework Version 1.7 at:
	
	http://blackboxframework.org/stable/blackbox-1.7.zip

and extract it into the "Blackbox" directory (you may have copied this directory elsewhere before).

Then, run "Blackbox/BlackBox.exe", open this file itself in the BlackBox framework (Blackbox/Duniter/Docu/MakingOfWWServerML.odc) and execute the commands below, by clicking on the successive following black-disk icons (when small windows open, click on OK). Then read the last lines at the bottom of this text.
