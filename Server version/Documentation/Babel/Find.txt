BabelFind

DEFINITION BabelFind;

	IMPORT Dialog;

	VAR
		dial: RECORD 
			line, col, pos: INTEGER;
			byPos: BOOLEAN
		END;

	PROCEDURE Position;
	PROCEDURE LineColGuard (VAR p: Dialog.Par);
	PROCEDURE PosGuard (VAR p: Dialog.Par);

END BabelFind.

Used to locate a position in a text, given its line and column numbers. It's the way one can locate Babel error messages.

"StdCmds.OpenToolDialog('Babel/Rsrc/FindWin.odc','#Babel:MFind')"

VAR dial
Interactor for the Babel/Rsrc/FindWin window.

line: INTEGER
Line number; first line has number 1.

col: INTEGER
Column number; first column has number 1.

pos: INTEGER
Position; first position has number 1.

byPos: BOOLEAN
TRUE iff pos is used or FALSE if line and col are used.

PROCEDURE Position
Guard: TextCmds.FocusGuard
Locates the position defined by the position dial.pos (if dial.byPos is TRUE) or by the line number dial.line and the column number dial.col (if dial.byPos is FALSE) in the focus text. Used in the Babel/Rsrc/FindWin window.

PROCEDURE LineColGuard (VAR p: Dialog.Par)
PROCEDURE PosGuard (VAR p: Dialog.Par)
Guard procedures for the text fields of the Babel/Rsrc/FindWin window.
