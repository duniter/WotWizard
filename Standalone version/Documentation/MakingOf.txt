Making of WotWizard


Download the Blackbox Component Framework Version 1.7 at:
	
	http://blackboxframework.org/stable/blackbox-1.7.zip

and extract it into a new empty directory (let's call it "NewBB"). Copy the "Blackbox" directory into "NewBB" (merge all subdirectories and replace all files when asked).

Then, run "NewBB/BlackBox.exe", open this file (command "File -> Open") itself in the BlackBox framework (NewBB/Duniter0/Docu/MakingOf.odc) and execute the commands below, by clicking on the black-disk icons (when small windows open, click on OK). Then read the last lines at the bottom.
